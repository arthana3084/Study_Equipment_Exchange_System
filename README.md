# 📚 Study Equipment Exchange System

A web-based platform that enables students to donate, browse, and request study equipment. The system encourages resource sharing, reduces unnecessary purchases, and promotes sustainability within educational institutions.

---

## ✨ Features

- User Registration & Login
- Donate study equipment
- Browse equipment by category
- Request available items
- View donated equipment
- Track user activities
- MySQL database integration

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- MySQL

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/arthana3084/Study_Equipment_Exchange_System.git
```

Install dependencies

```bash
npm install
```

Configure your MySQL database.

Update the database credentials inside:

```
db/connection.js
```

Run the server

```bash
node server.js
```

Open

```
http://localhost:3000
```
